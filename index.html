About this 3D map

A 3D interactive galaxy map of the crowd-funded and currently being developed game Star Citizen. Still in early alpha stage!

Created by Daughter of Sol [Shiari] on the RSI forums using WebGL. This map was announced in this forum thread.

The sourcecode and data that drives this map is available with the MIT license on my GitHub project page, and aside from my own code uses three.js, jQuery and jQuery UI.
